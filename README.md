#  Space Shooter - CS50 Week 0 (Scratch)

## Project Description

Space Shooter is a simple arcade-style game created using Scratch for **CS50's Introduction to Computer Science - Week 0**.

The player controls a spaceship that moves left and right using the arrow keys. Press the **Spacebar** to fire projectiles and destroy falling aliens. Each alien destroyed increases the score. If an alien collides with the spaceship, the player loses a life. The game ends when all lives are lost.

## Features

*  Spaceship controlled by the keyboard
*  Projectile firing using the Spacebar
*  Randomly falling aliens
*  Score tracking
*  Lives system
*  Game Over when all lives are lost

## Controls

| Key         | Action     |
| ----------- | ---------- |
| Left Arrow  | Move Left  |
| Right Arrow | Move Right |
| Spacebar    | Shoot      |

## Variables

* **Score** – Tracks the number of aliens destroyed.
* **Lives** – Tracks the player's remaining lives.

## Programming Concepts Used

This project demonstrates the following Scratch programming concepts:

* Events
* Variables
* Loops
* Conditional Statements (`if`)
* Broadcast Messages
* Cloning
* Collision Detection
